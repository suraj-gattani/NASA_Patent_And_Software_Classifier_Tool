The Information Sharing Protocol (ISP) VCR (ISPVCR) is a software program written in Tcl/Tk that provides a capability to record and playback ISP data via a Source Independent Telemetry Format (SITF) files. The ISPVCR provides a Graphical User Interface (GUI) that allows for start/stop of the crcording capability, specify output file names and start/stop of the playback of the SITF data file.
