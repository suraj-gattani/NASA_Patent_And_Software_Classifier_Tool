The Information Sharing Protocol (ISP) Logger (ISPLOGR) is a software program written in "C" that provides a capability to record ISP data into a Source Independent Telemetry Format (SITF) file. The ISPLOGR uses command-line arguments to specify output file names and other required parameters. Once the SITF file is defined, it can be used as input to other available ISP software available with for ISP data playback.
